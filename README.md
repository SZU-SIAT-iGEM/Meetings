# Meetings
作为SZU-SIAT-iGEM2026的会议记录仓库，存放会议记录，记录每一次或大或小的线上和线下讨论通讯。
## 📁 02_会议记录

### 用途说明

存放 **所有团队会议的记录**，确保讨论内容和决定事项有据可查。

### 目录结构

```
02_会议记录/
├── 📁 全体会议/
│   ├── 2026-01-15_全体会议.md
│   └── ...
├── 📁 组内会议/
│   ├── 📁 湿实验组/
│   ├── 📁 干实验组/
│   ├── 📁 HP组/
│   └── 📁 美工组/
├── 📁 外部会议/
│   ├── 2026-01-20_导师交流.md
│   └── ...
└── 📄 README.md
```

### 操作指南

| 会议类型 | 记录人 | 存放位置 | 发布时限 |
|----------|--------|----------|----------|
| 全体会议 | 轮值记录人 | `全体会议/` | 会后24小时内 |
| 组内会议 | 各组指定人员 | `组内会议/对应组/` | 会后24小时内 |
| 导师/外部会议 | 参会代表 | `外部会议/` | 会后24小时内 |

**会议记录命名格式**：`YYYY-MM-DD_会议主题.md`

**会议记录模板**：使用 `templates/会议记录模板.md`

### 会议记录应包含

- 会议基本信息（时间、地点、参会人员）
- 议程与讨论内容
- **决定事项**（加粗标注）
- **待办事项**（指定负责人和截止日期）
- 下次会议安排

### 自由扩展

- `📁 头脑风暴/` - 非正式讨论记录
- `📁 培训记录/` - 内部培训的记录
